# Mentoria Hub Telegram Bot

Это код для Telegram бота для Mentoria Hub.
Он отправляет напоминания о дедлайнах и (ВОЗМОЖНО, ЕСЛИ УСПЕЕМ ДОБАВИТЬ:) уведомления о новых возможностях 

## Установка

1. Установи Python
2. **Запусти бота**:
```bash
python main.py
```

## Как работает

- **Приветствие** (`/start`) — объясняет что умеет делать бот
- **Подписка** (`/subscribe`) — пользователь подписывается на уведомления
- **Уведомления** — каждый час проверяются дедлайны и отправляются напоминания
