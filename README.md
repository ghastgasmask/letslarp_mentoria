# Mentoria Hub

Платформа, которая собирает олимпиады, конкурсы, гранты и курсы для самостоятельного обучения для учеников 8–11 классов из Казахстана (и не только), с персональными рекомендациями на основе класса, интересов и целей.

Идея простая: большинство школьников даже не узнают о половине возможностей, на которые реально могли бы претендовать, потому что информация разбросана по Telegram-каналам, школьным объявлениям и сарафанному радио. Mentoria Hub собирает всё это в одном месте с поиском, плюс даёт учебный путь, чтобы реально подготовиться.

Сделано на [название хакатона] примерно за 4 дня.

## Что реально работает

- Регистрация и вход через Supabase, в два шага (сначала данные для входа, потом класс/интересы/имя)
- Роли — у студентов обычное приложение, у админов отдельная панель
- Каталог возможностей с фильтрами по категории, классу и формату, кнопка «в избранное»
- Страницы курсов с уроками, видео, прикреплёнными файлами и тестами по каждому уроку, результаты сохраняются
- Личный кабинет ученика — сохранённые возможности, курсы с прогрессом, ближайшие дедлайны
- Админ-панель для добавления/редактирования/удаления возможностей (курсы пока добавляются напрямую через Supabase)
- AI-ассистент в чате на базе Groq (llama-3.1-8b-instant), отвечает на вопросы про курсы/олимпиады/планы подготовки
- Лидерборд, календарь и роадмап ( без реальных данных, СТАТИЧНЫЕ МАКЕТЫЫЫЫ)
- Отдельный Telegram-бот для уведомлений — пока обрабатывает `/start`, `/subscribe`, `/unsubscribe`. Логика реальных напоминаний о дедлайнах ещё не подключена, это следующий шаг.

## Что пока замокано

Говорим честно

- Возможности в каталоге добавлены вручную, не парсятся в реальном времени. Рассматривали автоматический сбор из Telegram-каналов, но решили что за отведённое время это не сделать безопасно и надёжно, поэтому остановились на курируемых данных.
- Контент курсов (текст уроков, видео) это данные, которые мы написали сами, не от реальных преподавателей.
- Очки в лидерборде пока не считаются автоматически от прохождения курсов — таблица есть, но обновляется вручную.
- Telegram-бот шлёт приветствие и умеет регистрировать подписчиков, но пока не проверяет реальные дедлайны в Supabase и не пингует пользователей. Логика cron/задания в коде есть как заготовка, но не подключена.

## Стек

Фронтенд на React с Vite, стили через Tailwind, роутинг через React Router. Бэкенд это 
Supabase: авторизация, база данных Postgres, row-level security и хранилище файлов для видео/вложений к урокам. 
AI-ассистент работает через serverless-функцию на Vercel, которая обращается к API Groq. 
-бот — отдельный сервис на Python с использованием `python-telegram-bot`, не задеплоен вместе с веб-приложением!!!!!!!!!!!!!!

|---|---|
| Фронтенд | React + Vite, Tailwind CSS, React Router, иконки Lucide |
| Бэкенд / БД | Supabase (Postgres, Auth, Storage, RLS) |
| AI | Groq API, модель `llama-3.1-8b-instant`, вызывается из `/api/ai` |
| Бот | Python, `python-telegram-bot` |
| Хостинг | Vercel (фронтенд + serverless-функция) |

## Запуск локально (НЕ ДЕЛАЙТЕ, НЕ РЕКОМЕНДУЕТСЯ)

Понадобится установленный Node.js (проверь через `node --version`, если выдаёт ошибку — значит не установлен).

```bash
cd mentoria-hub
npm install
npm run dev
```

Откроется на `localhost:5173`. Нужно подключить свой проект Supabase — скопируй `src/lib/supabase.example.js` в `src/lib/supabase.js` и вставь свои Project URL и anon key из настроек API в Supabase.

Для бота:

```bash
cd letslarp_mentoria_bot
pip install -r requirements.txt
cp .env.example .env   # токен бота: 8962262440:AAGJnmruveyNePaAmo21MzO0TyksnH1Wwzk
python main.py
```

## База данных

Основные таблицы: `profiles` (расширяет `auth.users` из Supabase полями класс, интересы, роль), `courses` и `lessons`, `course_progress`, `opportunities` и `saved_opportunities`, `quizzes` и `quiz_results`, `leaderboard`. RLS включён на всех таблицах с пользовательскими данными, чтобы каждый видел и менял только свои строки. Полная схема — в SQL-скриптах, которыми мы всё это настраивали, сюда не включаем (это разовые setup-скрипты), но можем прислать по запросу.

Также есть публичный бакет в Supabase Storage под названием `uploads` для видео и файлов к урокам.

В общем, ОЧЕНЬ МНОГО разных таблиц в супабейз. Если хотите посмотреть, напишите нам!
