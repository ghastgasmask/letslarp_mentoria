# Промпт для Claude Code — Mentoria Hub

---

## Задача

Создай полноценный React-проект (Vite + React + Tailwind CSS) под названием **Mentoria Hub** — образовательная платформа для казахстанских школьников 8–11 классов. Проект должен выглядеть профессионально, чисто и современно в бело-синей цветовой гамме.

---

## Стек

- **React + Vite** (создай через `npm create vite@latest mentoria-hub -- --template react`)
- **Tailwind CSS v3** (подключи через `npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p`)
- **React Router DOM v6** (`npm install react-router-dom`)
- **Lucide React** для иконок (`npm install lucide-react`)
- Никаких других зависимостей пока не нужно

## Navbar (`src/components/Navbar.jsx`)

Фиксированный сверху (`fixed top-0`), белый фон, тонкая нижняя граница (`border-b border-neutral-200`), высота 64px, `z-50`.

**Левая часть** (логотип + навигация):
- Логотип: синий жирный текст «Mentoria» + маленький текст «Hub» рядом
- Навигационные кнопки (текст + иконка Lucide):
  - 📚 Курсы → `/courses`
  - 🔍 Возможности → `/opportunities`
  - 🏆 Лидерборд → `/leaderboard`
  - 📅 Календарь → `/calendar`
  - 🗺️ Роадмап → `/roadmap`

Используй иконки Lucide: `BookOpen`, `Search`, `Trophy`, `Calendar`, `Map`

**Правая часть**:
- Аватар профиля (круглый, 36x36px): использует `profile_placeholder.png` либо серый круг с инициалами «АИ» (заглушка). Рядом текст username: **«Айдана»**. Кликабельно → `/dashboard`
- Иконка `Settings` (Lucide, 22px) → `/settings`. Небольшой отступ слева от аватара.

Активный роут в navbar — подсвечивай синим текстом и тонкой синей линией снизу кнопки.

## ChatButton (`src/components/ChatButton.jsx`)

Фиксированная кнопка внизу справа (`fixed bottom-6 right-6`).
- Круглая, 56x56px, `bg-primary-600`, белая иконка `MessageCircle` из Lucide, тень (`shadow-lg`)
- При наведении — `bg-primary-700`
- Кликабельно → `/ai-assistant`
- Присутствует на всех страницах

## Общие требования к качеству

- Все страницы должны выглядеть **завершёнными**, не как заглушки. Никаких «Coming soon» или пустых белых экранов.
- Используй **реалистичные mock-данные** везде (имена, даты, описания на русском языке)
- **Адаптивность**: все сетки через Tailwind responsive классы (`sm:`, `md:`, `lg:`)
- Карточки с **тенью** (`shadow-sm` или `shadow-md`) и **скруглёнными углами** (`rounded-xl`)
- Hover-эффекты на кнопках и карточках (`hover:shadow-md transition-all`)
- Шрифт подключи через Google Fonts — **Inter** (добавь в `index.html`)
- `index.css` должен содержать `@tailwind base/components/utilities` и подключение Inter
